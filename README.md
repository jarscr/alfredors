Hola 👋 Mi nombre es Alfredo Rodríguez Siles
==================================Desarrollador Web-----------------Cuento con 20 años de experiencia en programación PHP, 2 años en programacion C#, conocimientos en SQL Server, administración de servidores Linux y servidores de red. Me especialice en PHP hace mas de 15 años, que en principio era programación básica lo que realizaba. Actualmente estoy desarrollando un sistema de Punto de Ventas que corre en la nube y también corre en Windows como un servicio conectado con C# y PHP. Se ha desarrollado aplicativos B2B, Sistemas de Recursos Humanos, Comercio Electrónico, sitios web multilenguaje y algunos modulos para WordPress y Prestashop. Conectividad con los sistemas de pago en línea con Bac Credomatic, Banco Nacional de Costa Rica, Banco de Costa Rica, Primerica, Credix, BBVA, Alignet Perú. Cuento con un conector al API de Shopify para Factura Electrónica de Costa Rica, también un sincronizador de inventarios y con WooComerce basado en un ERP con base de datos en lenguajes SQL.* 🌍  Soy de  Costa Rica* 🖥️  Mi sitio web es [Sitio Web](http://www.jarscr.com)* ✉️  Si requiere contacterme puedes escribir a [alfredo@jarscr.com](mailto:alfredo@jarscr.com)* 🚀  Actualmente estoy trabajando en [MiPOS](http://www.mipos.co.cr)* 🧠  Estoy estudiando C# & Flutter
